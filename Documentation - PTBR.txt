Interact System (Inspect)
Localização

Blueprints: Content/Blueprints/

Configurações/auxiliares: Content/Others/

Blueprint principal

InspectMaster_BP

Criar um item inspecionável

Clique com o botão direito em InspectMaster_BP → Create Child Blueprint Class

Use o Child Blueprint criado como o “novo item” inspecionável do seu projeto.

Configuração do item (Details)

No Child Blueprint, ajuste em Details:

Static Mesh: defina a malha do item a ser inspecionado.

Item Description: configure Nome e Descrição do item.

Widget usado

ItemDescription-WG — widget que exibe as informações do item durante a inspeção.

Interact System
Ativar interação em um Blueprint

No Blueprint do item que você quer interagir:

Abra Class Settings

Em Details → Interfaces → Implemented Interfaces, adicione: HR_InteractInterface

Evento de interação

No próprio Blueprint do item, adicione o evento Event Interact (é o evento chamado pela interface).

Exemplo

Content/Blueprints/Others/Test_BP — exemplo de implementação do sistema de interação.

Configurações gerais

Content/Blueprints/HR_Character_BP — configurações principais do Interact System.
