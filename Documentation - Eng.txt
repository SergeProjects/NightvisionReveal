Interact System (Inspect)
Location

Blueprints: Content/Blueprints/
Settings/helpers: Content/Others/

Main Blueprint

InspectMaster_BP

Create an inspectable item

Right-click InspectMaster_BP → Create Child Blueprint Class
Use the created Child Blueprint as the new inspectable item for your project.

Item setup (Details)

In the Child Blueprint, set in Details:
- Static Mesh: set the item mesh to be inspected.
- Item Description: configure the item Name and Description.

Widget used

ItemDescription-WG — widget that displays the item information during inspection.

Interact System
Enable interaction on a Blueprint

On the Blueprint you want to interact with:
- Open Class Settings
- In Details → Interfaces → Implemented Interfaces, add: HR_InteractInterface

Interaction event

On the same Blueprint, add the Event Interact node (the interface event that gets called).

Example

Content/Blueprints/Others/Test_BP — example implementation of the interaction system.

General settings

Content/Blueprints/HR_Character_BP — main Interact System settings.

System logic

Content/Blueprints/HR_Character_BP — contains the logic to grab/hold boxes.

Make an object grabbable

- Place a Static Mesh (or an actor with a mesh/box) in the level.
- In the component Details, enable Simulate Physics.
